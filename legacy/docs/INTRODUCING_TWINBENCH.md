# Introducing TwinBench

TwinBench is the open benchmark for personal AI assistants.

It exists because current AI benchmarks still miss the gap between a chatbot and a real assistant that remembers, acts, follows up, stays safe, and operates over time.

TwinBench is designed to be:

- easy to understand
- easy to run
- easy to challenge
- hard to game

The benchmark is evidence-first. It separates verified score from projected score, records measured coverage, and keeps degraded runs visible instead of quietly deleting them.

Nullalis is the current reference runtime because it produced the first strong public artifact. That does not make Nullalis the owner of the benchmark. It makes TwinBench a real public challenge.

The goal is simple:

When teams say they are building personal AI assistants, TwinBench should be the benchmark they want to pass.
