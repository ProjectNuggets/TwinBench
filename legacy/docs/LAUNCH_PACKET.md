## One-Paragraph Benchmark Description

TwinBench is the open benchmark for personal AI assistant runtimes. It measures whether a runtime can behave like a real personal AI assistant: remember, act, follow up, stay safe, and operate over time. The benchmark is evidence-first, vendor-neutral, and designed for real runtime systems rather than one-shot prompt demos.

## FAQ Answers

### Is this a chatbot benchmark?

No. TwinBench is for long-lived assistant runtimes, not one-shot conversation quality.

### Is this only for Nullalis?

No. Nullalis is the first reference runtime artifact. The benchmark is meant to be challenged publicly by other teams.

### What if my runtime is only partially compatible?

That is still useful. TwinBench distinguishes native compatibility, bootstrap limits, adapter needs, and partial runs instead of flattening everything into a single bad score.
