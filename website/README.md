# TwinBench Website

This directory is the static public surface for TwinBench.

It is generated from checked-in artifacts and assets with:

```bash
make site
```

Key goals:

- make TwinBench legible in under 10 seconds
- provide a public leaderboard surface
- give every result a shareable URL
- keep the benchmark story cleaner than a raw repo index

Generated pages include:

- homepage
- methodology
- FAQ
- submit page
- compare page
- per-result pages
