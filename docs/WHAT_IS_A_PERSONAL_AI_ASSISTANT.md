# What Is a Personal AI Assistant?

A personal AI assistant is not just a chatbot.

It is a long-lived system that:

- remembers relevant user context over time
- follows up without losing the thread
- executes tasks instead of only answering
- stays safe during background or autonomous behavior
- behaves consistently across time and surfaces

That is why TwinBench uses two public phrases:

- **personal AI assistants**
- **personal AI assistant runtimes**

The first phrase is public and legible.
The second phrase is useful when talking to builders and runtime teams.

If a system cannot operate over time, it may still be a great model or assistant product. It is just not yet the full category TwinBench is trying to define.
